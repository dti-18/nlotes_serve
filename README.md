# nlotes_serve
Template para proyectos de DTI 2018

## Features

- Python 3.6 runtime environment. 

Install requirements::

    $ pip install pipenv

Install dependencies with pipenv::

    $ pipenv install
    $ pipenv shell

Runing::

    $ (nlotes_serve-u3riQcTX) D:\dev\dti-a\nlotes_serve>manage.py runserver 80
